SYSTEM INDEX // SECURE TRANSMISSION PROTOCOL

This system secures any file into a custom **.cyphr** format using a user-defined password key. **Transmission is secured only by the secret password.**

OPERATIONAL INSTRUCTIONS

1

CREATE THE FILE (Sender)

Go to the **Encryption Terminal**. Enter a **strong, shared password** and upload your file (any type: Image, Video, or Text). Download the custom **.cyphr** file.

2

TRANSMIT

Send the **.cyphr** file and the **secret password** to the recipient separately (e.g., file via email, password via chat).

3

DECRYPT AND ACCESS (Recipient)

Go to the **Decryption Terminal**. Upload the **.cyphr** file and input the **exact password** to reveal and display the original content.
